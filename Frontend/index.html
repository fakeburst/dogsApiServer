/api/dog/image

Parameters

Name Values Required Description
age old / puppy / young No Age of dog to return.
breed Possible breeds No Breed of dog to return.
name name No The name of dog will be included in response.
raw src / html No The output format, as url or html img tag. By default -- JSON.

Example: This'll return you json data for random old poodle with random name.

http://dogsapi-fakeburst.rhcloud.com/api/dog/image?breed=poodle&age=old&name=name

/api/dog/name

Parameters

Name Values Required Description
gender male / female / all No Gender of dog name to return. By default -- all.

Example: This'll return you a random female dog name.

http://dogsapi-fakeburst.rhcloud.com/api/dog/name?gender=female
